# JavaScript Learning

Personal repository for learning JavaScript, from browser to server (Node.js).

## What's Inside

- **Fundamental JavaScript** — variables, functions, loops, etc
- **DOM** — manipulating HTML elements with JavaScript
- **Asynchronous** — callback, promise, async/await
- **Array & DSA** — array methods
- **Node.js** — JavaScript on the server side
- **Express.js** — building APIs
- **MongoDB** — database for storing data
